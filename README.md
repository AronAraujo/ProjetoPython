# 📁 Sistema de Cadastro em Python

## 🚀 Descrição
Aplicação desenvolvida em Python para cadastro e gerenciamento de dados via terminal, com persistência em arquivo `.txt`.

O sistema permite adicionar, visualizar e armazenar informações de forma simples, simulando um banco de dados básico sem uso de SGBD.

---

## 🛠 Tecnologias utilizadas
- Python 3
- Manipulação de arquivos (.txt)

---

## ⚙️ Funcionalidades
- Cadastro de novos registros
- Armazenamento em arquivo `.txt`
- Leitura e exibição dos dados salvos
- Interface via terminal
